ARMAZENAMENTO

Armazenamento

Descrição: Local para guardar dados de forma organizada, permitindo o acesso posterior.

Arquivo

Descriçãp: coleção de dados armazenados sob um nome único.

Backup

Descrição: Backup é uma cópia de segurança de dados essencial para proteção.

Banco de Dados

Descrição: sistema organizado para armazenar, gerenciar e recuperar informações.

Bluetooth

Descrição: comunicação sem fio entre dispositivos.

Compactar

Descrição: processo de redução do tamanho de um arquivo ou pasta.

Criptografia

Descrição: proteção de informações por de ténicas de codificação.

Dados

Descrição: informações manipuláveis em formato digital ou analógico.

Decodificação

Descrição: processo de reverter a codificação ou criptografia aplicada aos dados.

Descompactar

Descrição: restaurar o arquivo ou pasta à sua forma original após ter sido compactado.

Disquete

Descrição: meio de armazenamento magnético removível, amplamente utilizado nas décadas anteriores.

Documento

Registro ou representação de informações, frequentemente em formato escrito ou eletrônico, organizado de maneira a comunicar ou armazenar dados.

Download

Descrição: obter dados ou arquivos da internet para armazenamento no dispositivo local.

Extensão de Arquivo

Descrição: sequência de caracteres no final de um nome de arquivo que indica o tipo do arquivo.

PDF

Descrição: formato de arquivo projetado para apresentar documentos de maneira independente de dispositivo ou software.

Repositório

Descrição: local ou sistema onde algo é armazenado.

Servidor

Descrição: computador dedicado ao armazenamento e gerenciamento de dados.

Upload

Descrição: enviar dados ou arquivos do dispositivo local para um servidor ou plataforma online.
