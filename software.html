SOFTWARE

Software

Descrição: parte não física de um computador.

API

Descrição: conjunto de regras que permite que aplicativos se comuniquem entre si.

Aplicativo

Descrição: programa de software projetado para realizar tarefas específicas em dispositivos eletrônicos.

Atualização

Descrição: modificação feita no Software com o intuito de melhorar o sistema.

Backup

Descrição: Backup é uma cópia de segurança de dados essencial para proteção.

Código Fonte

Descrição: instruções de um programa compreensíveis por programadores.

Compilador

Descrição: converte o código-fonte em instruções executáveis.

Criptografia

Descrição: proteção de informações por de ténicas de codificação.

Desenvolvedor

Descrição: criador de softwares e de aplicações.

Framework

Descrição: estrutura para facilitar o desenvolvimento de software.

Integração

Descrição: união harmoniosa de sistemas ou componentes.

Interface

Descrição: forma de interação com um dispositivo ou programa.

Manutenção

Descrição: ajustes e melhorias contínuas no software.

Multitarefa

Descrição: capacidade de um sistema realizar várias tarefas simultaneamente.

Plataforma Virtual

Descrição: ambiente simulado para executar software.

Processamento

Descrição: execução de operações em dados ou informações.

Programa

Descrição: conjunto de instruções que realiza uma tarefa específica em um computador.

Requisitos

Descrição: especificações necessárias para desenvolver ou utilizar um software.

Sistema Operacional

Descrição: software que controla e gerencia um computador.
